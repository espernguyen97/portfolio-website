Project Details

(San Francisco International Airport (SFO)'s Air Traffic Analysis)

Overview:

Objectives:

Create Tableau charts and the dashboard to analyze the air traffic data of San Francisco International Airport in 2020:
Task 1: A visualization that shows the total number of flights at San Francisco International Airport.
Task 2: Create a chart that depicts the number of flights taking off per day in March and then find the busiest day at San Francisco International Airport.
Task 3: Create a line chart that shows the total number of flights per day in a month at San Francisco International Airport.
Task 4: Create a bar chart that represents the top 10 busiest flight routes.
Task 5: Create a bar chart that shows the top 10 longest flight routes.
Task 6: Create a map that represent the flights from San Francisco International Airport to different parts of the world.
Task 7: Create a dashboard of the charts and map made for all tasks above.
