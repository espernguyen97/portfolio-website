Project Details

(2020's Covid-19 Data Analysis)

Overview:

Goals:

Task 1: A visualization that shows the total cases, total death, global death percentage in 2020.
Task 2: Compares the total death in different continents.
Task 3: Compares the percentage of population infected in different countries.
Task 4: Compares the percentage of population infected per month in different countries.
Task 5: Create a dashboard of the charts made for all tasks above.



Reference Resources:




After inserting global covid-19 data into SQL Server:






Task 1:

A visualization that shows the total cases, total death, death percentage in 2020 in the world.

Next, Inserting the data retrieved from SQL Server into Tableau
