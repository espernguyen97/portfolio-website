About AppServ :

The Idea: Help connect Clients and Services they need anywhere, anytime.
This web app’s premise and design aim to create a live access "search" that directs users to an immediate search engine solution for a specific service.

Goals:

The web app should allow the customers to view a specific service nearby either through the map or by typing in a name and choosing from a service list in the search bar, and to book based on users’ current locations and available time slots.
